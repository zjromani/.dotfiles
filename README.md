### Sym Links

```bash
ln -s /Users/zachromani/.dotfiles/iterm_profile /Users/zachromani/iterm_profile

ln -s /Users/zachromani/.dotfiles/psqlrc /Users/zachromani/.psqlrc

ln -s /Users/zachromani/.dotfiles/vimrc /Users/zachromani/.vimrc
```
